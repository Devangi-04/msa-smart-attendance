## üìù Method 2: Convert to Word Document

### Option A: Copy-Paste to Word
1. Open `SYNOPSIS.html` or `SRS.html` in browser
2. Press `Ctrl + A` to select all
3. Press `Ctrl + C` to copy
4. Open Microsoft Word
5. Press `Ctrl + V` to paste
6. Save as `.docx`

### Option B: Use Online Converter
1. Go to: https://cloudconvert.com/html-to-docx
2. Upload `SYNOPSIS.html` or `SRS.html`
3. Convert to DOCX
4. Download the Word document

## üîß Method 3: Using Pandoc (Advanced)

### Install Pandoc:
- **Windows**: Download from https://pandoc.org/installing.html
- **Mac**: `brew install pandoc`
- **Linux**: `sudo apt-get install pandoc`

### Convert Commands:
```bash
# Convert to PDF
pandoc SYNOPSIS.md -o SYNOPSIS.pdf
pandoc SRS.md -o SRS.pdf

# Convert to Word
pandoc SYNOPSIS.md -o SYNOPSIS.docx
pandoc SRS.md -o SRS.docx
```
